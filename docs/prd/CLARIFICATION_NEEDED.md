## 3. 거래 로직 관련 질문

### 3.1 진입값과 손절값 관계
**질문:** 진입 후 손절 조건이 `kimchi >= 손절값`인데, 진입값과 손절값의 관계는?

**예시 시나리오:**
- 진입값: 3.0%
- 손절값: 4.0%
- 현재 kimchi: 3.5%

이 경우:
- 3.5% >= 3.0% → 진입 조건 충족
- 진입 직후 3.5% >= 4.0%가 아니므로 손절 안됨
- **손절값은 진입값보다 항상 높아야 하는가?**
